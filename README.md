# NUXT Offical WebSite

## Requirment

- Node Version v20.10.0

## 开发

```bash
    npm i
    npm run dev
```

## 部署

### 静态部署

- Node Version v20.10.0

```bash
npm i
npm run generate # .output文件夹 
# .output/public  or dist 
# You can now deploy .output/public to any static hosting!

```

## pm2 部署

```bash
npm i
npm run build
#.output
npm run start
```
